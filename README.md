- **Security & AI** — alert triage, threat hunting, detections, and building agentic systems that can reason, retrieve, and act.
- **Web3** — smart contracts, DApps, secure Solidity patterns, Hardhat, ecosystem tooling.
- **Frontend engineering** — React, Next.js, TypeScript, accessible UI, performance.
- **Full stack** — Node.js, Python, APIs, databases, automation.
- **Data** — analytics, BI-style storytelling, charts & dashboards.
- **Research** — blockchain, LLM/AI evaluation, and whatever breaks next in security.
